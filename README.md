### LumenRoute

**Project Manager** · Mobile Safety Navigation App · [Repository](link-to-repo) <br>
**Tech Stack:** React Native / Expo / Google Maps API / OpenStreetMap API / Firebase / BeautifulSoup / Firecrawl / News API

<details>
<summary><strong>Safety-based navigation app that ranks routes by safety and ETA</strong></summary>

<br>

LumenRoute is a mobile navigation app that helps users choose routes that balance **safety** and **estimated travel time**. Instead of only optimizing for the fastest route, LumenRoute generates route options and ranks them from safest to most unsafe based on a custom safety index.

The safety score is calculated using historical crime data, including public data from the **Berkeley Police Department**, along with mapping APIs and route information. Our team designed a graph-based route evaluation system that accounts for factors such as crime location, recency, severity, and route proximity to incidents.

**Project Highlights**

- Led project planning, team coordination, and feature prioritization across product, design, and development.
- Helped define the core product direction: route recommendations that balance **safety, ETA, and usability**.
- Supported development of a custom safety-ranking formula using crime severity, recency, location, and route exposure.
- Coordinated work across mapping APIs, public crime datasets, Firebase, and real-time information sources.
- Helped shape user-facing safety features, including route ranking and friend check-ins.

**Key Features**

- Ranked route options from safest to most unsafe
- Safety index based on historical crime data, severity, recency, and location
- Graph-based route evaluation system
- Integration with Google Maps and OpenStreetMap APIs
- Berkeley PD public crime data integration
- Real-time local safety/news updates using scraping and news APIs
- Friend check-in feature for sharing route status and safety updates

</details>
